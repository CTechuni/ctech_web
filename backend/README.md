# backend con FastAPI y PostgreSQL

Este es el backend del proyecto que usa:
FastAPI (framework principal)
PostgreSQL (base de datos)
SQLAlchemy (ORM)
Uvicorn (servidor)

## Ejecutar el servidor
uvicorn app.main:app --reload
fastapi den main.py desde la terminar dentro de la carpeta ./app/

Dependencias principales
fastapi
uvicorn
sqlalchemy
psycopg2-binary
python-dotenv
